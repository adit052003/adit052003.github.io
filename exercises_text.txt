Exercise: Simple A-frame example
2024-05-23
Phillip G. Bradford
Goal: update text and position of text on Laptop and see effects in VR glasses
Exercise
This example is running A-frame from a basic HTML file – good to start with not-index.html
In a web-browser such as google-chrome
Select <CTRL>-o, then open the file simple-sky.html
Move the scene around using the mouse-arrow
Update the text in simple-sky.html and save
Refresh the file simple-sky.html in the browser
Verify the image has been updated in your phone using the VR glasses
Change the a-text position values
Do one at a time – change them by small increments such as 1 or 2; In the case of z update by 5 or 10.
Use VR glasses to view the scene given these changes

Exercise: Forrest image
2024-01-08
Phillip G. Bradford
Goal: use a-scene, a-sky tags and basic geometric objects and a static image
Exercise
Find an alternative (free and public-domain) image like the forest image or nebula image showing <a-scene>…</a-scene>, <a-sky>…</a-sky> and <a-text>…</a-text>
Find a free and public domain 2D image (most images are 2D). Or you may use the forest image in this repo
For example: https://images.google.com/ 
Download an image to your laptop
Upload the image into your project in glitch
Get glitch-assigned URL to the image you just uploaded and point src=”glitch-url” in <a-sky>…</a-sky>  component of “forest.html”
Update the words in <a-text>…</a-text>
Work with the screen from <CTRL>-<ALT>-I on Windows and <CMD>-<OPTION>-I on MacOS
On the phone instance of your short-git repository, click the new glitch URL
Use VR glasses to view the scene
